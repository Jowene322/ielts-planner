# 雅思7分冲刺计划网页版 App

这是一个为你的雅思备考定制的单文件网页应用，支持每日任务展示、打卡、积分、连续打卡、日历视图、学习数据统计，以及在缺勤后动态重新安排学习计划。

---

## 📌 你的备考档案

- **当前基础**：四级 493 / 六级 503（约雅思 5.5–6.0）
- **已背词汇**：《刘洪波雅思词汇真经》1000+/3000+
- **已完成**：剑雅 5 Test 1 听力精听
- **目标分数**：总分 7.0（纸笔考试）
- **考试日期**：2026年8月29日
- **每日投入**：3–4 小时，集中在上午

**分数策略**：听力 7.5 + 阅读 7.5 + 写作 6.5 + 口语 6.5 = 总分 7.0

## 🚀 如何使用

### 在电脑上打开

直接用浏览器打开 `index.html` 文件即可，推荐 Chrome / Edge / Safari。

```
双击 index.html
```

### 在手机上打开（三种方式）

#### 方式一：免费部署到 GitHub Pages（推荐）

1. 注册 [GitHub](https://github.com) 账号；
2. 新建一个仓库，例如 `ielts-planner`；
3. 把 `index.html` 上传到仓库；
4. 进入仓库 Settings → Pages → Source 选择 main 分支；
5. 获得一个 `https://你的用户名.github.io/ielts-planner` 链接；
6. 手机浏览器访问这个链接即可，还可以添加到手机主屏幕像 App 一样使用。

#### 方式二：通过微信/QQ发送给自己

1. 把 `index.html` 发送到微信“文件传输助手”；
2. 在手机上点击文件，选择“用浏览器打开”；
3. 部分安卓手机可以直接打开，iPhone 可能需要配合“文件”App 预览。

#### 方式三：本地局域网访问

1. 确保手机和电脑连接同一个 Wi-Fi；
2. 在电脑上进入 `ielts-planner` 文件夹；
3. 用 Python 启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
4. 查看电脑的内网 IP（如 `192.168.1.5`）；
5. 手机浏览器访问 `http://192.168.1.5:8000`。

## ⭐ 打卡与积分规则

- **基础积分**：每完成 10 分钟任务获得 1 积分；
- **全勤奖励**：当天全部任务完成并打卡，额外 +20 积分；
- **连续打卡**：连续 3 天 +30 积分，连续 7 天 +100 积分；
- **缺勤处理**：某天未打卡会打断连续打卡；进入 App 时可选择重新安排后续计划。

---

## 🔄 缺勤后自动重排

如果你某天没有学习，App 会：

1. 检测所有未打卡的日期；
2. 将这些日期的未完成任务加入待办池；
3. 按优先级重新分配到后续日期（听力 > 阅读 > 写作 > 口语 > 词汇）；
4. 考前 3 天自动减少任务量，以复习为主；
5. 如果任务太多无法在考前完成，会提示你增加学习时长或精简任务。

## ⚠️ 重置数据

如果你想从头开始，可以点击“学习数据”页面的“重置所有数据”按钮。这会清空：

- 所有任务完成状态
- 打卡记录
- 连续打卡和积分

## 💡 备考建议

1. **听力和阅读是拉分关键**：这两科短期提分最快，优先保证练习量；
2. **写作一定要限时**：考前 4 周开始每篇 Task 2 必须严格 40 分钟完成；
3. **口语每天开口**：即使只有 15 分钟，也要录音或自言自语练习；
4. **模考要全真**：周六上午按真实考试时间做完整套题，培养生物钟；
5. **考前一周不做新题**：只看错题、词伙、模板和口语素材。

## 🛠 技术说明

- 单文件 HTML，无需构建工具；
- 使用 Tailwind CSS CDN 做响应式布局；
- 数据保存在浏览器 localStorage 中；
- 支持手机浏览器访问，推荐部署到 GitHub Pages 后添加到主屏幕。
