# 雅思7分冲刺计划（新版：7.10-8.28）

这是一个为你的雅思备考定制的单文件网页应用，支持每日任务展示、打卡、积分、连续打卡、日历视图、学习数据统计，以及在缺勤后动态重新安排学习计划。

---

## 📌 你的备考档案

- **当前基础**：四级 493 / 六级 503（约雅思 5.5–6.0）
- **已背词汇**：《刘洪波雅思词汇真经》1000+/3000+
- **已完成**：剑雅 5 Test 1 听力精听
- **目标分数**：总分 7.0（纸笔考试）
- **考试日期**：2026年8月28日
- **每日投入**：约 8 小时高强度学习（8:00–18:30），19:00–23:00 做实验

**分数策略**：听力 7.5 + 阅读 7.5 + 写作 6.5 + 口语 6.5 = 总分 7.0

## ⭐ 打卡与积分规则

- 完成任务获得基础积分（1积分/10分钟）
- 当日全部完成额外 +20 积分
- 连续打卡 3 天 +30 积分，7 天 +100 积分
- 缺勤打断连续打卡

---

## 🔄 缺勤后自动重排

如果某天没打卡，App 会：
1. 检测未打卡日期
2. 把积压任务按优先级（听力 > 阅读 > 写作 > 口语 > 词汇）分配到后续日期
3. 考前 3 天自动减少新任务

---

## ⚠️ 重要提醒

1. **每天8小时强度极高**，必须保证睡眠和营养
2. **每7天安排一次复盘/轻量日**，避免 burnout
3. **口语不要死记硬背**，要用个人素材自然表达
4. **考前3天不做新题**，只看错题和模板
5. **健身日适当减量**，但不可完全跳过
